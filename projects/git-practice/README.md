# Git Practice

## Core Commands

```bash
git init
git status
git add .
git commit -m "Describe the change"
git log --oneline
git branch
git switch -c new-branch
```

## First Practice Exercise

1. Create or edit a file.
2. Run `git status`.
3. Stage it with `git add`.
4. Commit it with a clear message.
5. Review the history with `git log --oneline`.

## Commit Message Examples

```text
Add cybersecurity learning roadmap
Document Fedora troubleshooting commands
Create first responsive web page
Update progress log for JavaScript lesson
```
